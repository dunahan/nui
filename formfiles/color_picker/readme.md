### Usage:

`NUI_DisplayForm(oPC, "color_picker");`

### Configuration

In `nui_c_colorpick.nss`, set the value of `COLOR_VARNAME` to the module variable name to set the selected color's json object to.  See comments in `nui_c_colorpick.nss`.
